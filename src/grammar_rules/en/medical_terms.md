# Medical Terminology Rules — Radiology (EN)

## Standard Plurals
Apply correct English medical plurals:
- vertebra → vertebrae
- diagnosis → diagnoses
- prognosis → prognoses
- calculus → calculi
- thrombus → thrombi
- embolus → emboli
- lumen → lumina
- septum → septa
- appendix → appendices
- cortex → cortices
- index → indices (in anatomical context)

## Capitalisation
- Anatomical proper names: **Achilles tendon**, **Circle of Willis**, **Fallopian tube**
- Imaging modalities in full: **MRI**, **CT scan**, **PET scan**, **SPECT**, **X-ray**

## Spelling Variants (use US or UK consistently — prefer US in medical reports)
- haemorrhage / hemorrhage → **hemorrhage**
- oedema / edema → **edema**
- tumour / tumor → **tumor**
- colour / color → **color**
- anaesthesia / anesthesia → **anesthesia**
- orthopaedic / orthopedic → **orthopedic**
- foetus / fetus → **fetus**

## Common Radiology Terms
- Use **lesion** (not "wound" or "injury") for focal abnormalities
- Use **opacity** for areas of increased density on X-ray/CT
- Use **enhancement** for post-contrast signal increase
- Use **heterogeneous** / **homogeneous** (not "heterogenous")
- Use **attenuating** (CT) / **signal** (MRI) — do not interchange
- **T1-weighted**, **T2-weighted** (hyphenated, with numeral)
- **coronal**, **sagittal**, **axial** planes — lowercase unless starting a sentence

## Abbreviations (expand on first use)
- CT — Computed Tomography
- MRI — Magnetic Resonance Imaging
- FDG — Fluorodeoxyglucose
- SUV — Standardised Uptake Value
- GRE — Gradient Echo
- FLAIR — Fluid-Attenuated Inversion Recovery
