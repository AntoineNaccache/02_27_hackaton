## Accord du participe passé

### Avec être : accord avec le sujet
- Elle s'est **crue** obligée. (sujet = elle → féminin)
- Ils sont **partis** tôt.
- Les alvéoles furent **brisées** (alvéole = féminin).

### Avec avoir : accord avec le COD si placé AVANT
- Les arrhes qu'ils avaient **données** (COD "arrhes" placé avant → féminin pluriel).
- Il les a **vus**.
- Pas d'accord si le COD suit : "Ils ont donné les arrhes."

### Verbes pronominaux
- Accord avec le sujet quand le pronom est COD :
  - Elle s'est **blessée**. (se = COD)
- Pas d'accord quand le pronom est COI :
  - Elle s'est **lavé** les mains. (se = COI, les mains = COD postposé)

## Accord des adjectifs

### Règle générale
- Masculin singulier : base (ex. : vieux, vieilli)
- Féminin singulier : + e (ex. : vieille, vieillie)
- Masculin pluriel : + s (ex. : brisés)
- Féminin pluriel : + es (ex. : brisées)

### Adjectifs invariables
- Les adjectifs de couleur composés ou noms de couleur : rose, crème, marron, etc.
- Les adjectifs en apposition désignant la teinte exacte.

## Accord du verbe avec le sujet

### Sujets collectifs
- "Un groupe de patients **présentent**" ou "**présente**" — les deux sont acceptés.
- "La plupart des lésions **sont**" → accord au pluriel.

### Sujet inversé
- "Furent brisés deux alvéoles" → accord avec "alvéoles" (pluriel, féminin → "brisées" ou "brisés" selon le genre reconnu).
