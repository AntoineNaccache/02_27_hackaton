# Formatting Guidelines — Radiology Report (EN)

## Mandatory Structure

The report must include the following sections, in this order:

---

### 1. HEADER
- Name of the institution
- Address / city of the institution
- Exam date (format: Month DD, YYYY) — use today's date unless a specific date is mentioned in the report

### 2. PATIENT INFORMATION
- Patient full name (if mentioned)
- Date of birth (if mentioned)

### 3. RESULTS
The input text, reproduced without any modification.

### 4. SIGNATURE
- First and last name of the signing physician (with title "Doctor")
- Signing location
- Signing date

---

## Writing Rules

1. **Absolute fidelity**: the RESULTS field must reproduce the input text without any modification, reformulation, or summary.
2. **Missing information**: if information is not available in the source text, use "[Not provided]" rather than inventing it.
3. **Date**: use today's date if no specific date is mentioned in the text.
4. **Capitalisation**: section titles in UPPERCASE, proper nouns with initial capital letter.
5. **Signature**: always include the title "Doctor" before the physician's name.

---

## Expected Layout Example

```
RADIOLOGY EXAMINATION REPORT
=============================
Institution : St. Mary's Hospital
City        : London
Exam date   : February 27, 2022

PATIENT INFORMATION
-------------------
Name          : [Not provided]
Date of birth : [Not provided]

RESULTS
-------
The patient presents a tibio-fibular fracture with bone loss.
A cast is recommended for six weeks with regular dressing changes.
The patient should be re-examined after eight months with an X-ray at St. Mary's Hospital.

                              Doctor John Smith
                              London, February 27, 2022
```
