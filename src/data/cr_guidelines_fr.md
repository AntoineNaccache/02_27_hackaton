# Directives de mise en forme — Compte Rendu Radiologique (FR)

## Structure obligatoire

Le compte rendu doit comporter les sections suivantes, dans cet ordre :

---

### 1. EN-TÊTE
- Nom de l'établissement
- Adresse / ville de l'établissement
- Date de l'examen (format : JJ mois AAAA) (use current date sauf si une date précise est mentionnée dans le compte rendu)

### 2. INFORMATIONS PATIENT
- Nom et prénom du patient (si mentionné)
- Date de naissance (si mentionnée)

### 3. RÉSULTATS
Le texte donné en entrée, sans aucune modification.

### 4. SIGNATURE
- Prénom et Nom du médecin signataire (avec titre « Docteur »)
- Lieu de signature
- Date de signature

---

## Règles de rédaction

1. **Fidélité absolue** : le champ RÉSULTATS doit reproduire le texte d'entrée sans aucune modification, reformulation ou résumé.
2. **Informations manquantes** : si une information n'est pas disponible dans le texte source, indiquer « [Non renseigné] » plutôt que d'inventer.
3. **Date** : utiliser la date du jour si aucune date précise n'est mentionnée dans le texte.
4. **Majuscules** : titres de section en MAJUSCULES, noms propres avec majuscule initiale.
5. **Signature** : toujours inclure le titre « Docteur » devant le nom du médecin.

---

## Exemple de mise en page attendue

```
COMPTE RENDU D'EXAMEN RADIOLOGIQUE
====================================
Établissement : Hôpital Saint-Antoine
Ville          : Aumont-Chantilly
Date           : 27 février 2022

INFORMATIONS PATIENT
--------------------
Nom            : [Non renseigné]
Date de naissance : [Non renseigné]

RÉSULTATS
---------
Le patient présente une fracture du tibia-péroné avec perte d'os.
Il faut savoir que dans ce cas-là, on recommande un plâtre pour six semaines avec des pansements réguliers.
Aussi, il faudra que le patient soit réexaminé au bout de huit mois avec une radiographie, à l'Hôpital Saint-Antoine.

                              Docteur Philippe Naccache
                              Aumont-Chantilly, le 27 février 2022
```
