# Règles sur les conjonctions et locutions complexes

## "Quelque" / "Quelles que" / "Quel que"

### Quel(le)(s) que — suivi du verbe être (ou verbe d'état)
- S'écrit en **deux mots**, s'accorde avec le sujet du verbe qui suit.
- **Quelles que soient** vos raisons... (sujet "raisons" = féminin pluriel)
- **Quels que soient** les résultats... (sujet "résultats" = masculin pluriel)
- **Quelle que soit** la cause... (sujet "cause" = féminin singulier)
- **Quel que soit** le motif... (sujet "motif" = masculin singulier)

### Quelque — adverbe (invariable, sens de "si...que")
- Suivi d'un adjectif + que : **quelque exiguës qu'aient pu paraître...**
- Ici "quelque" est adverbe et est **invariable**.
- Attention : si le nom suit directement, c'est l'adjectif "quelque" et il s'accorde :
  - "quelques raisons" (déterminant, pluriel)

### Quelqu' — élision devant voyelle
- "quelque" + voyelle → **quelqu'** : **quelqu'exiguës**, **quelqu'un**, **quelqu'une**
- L'élision est **obligatoire** devant une voyelle ou h muet.

## "Quoi que" / "Quoique"

### Quoi que — deux mots (= quelle que soit la chose que)
- **Quoi qu'il en soit**, c'est vrai. (= quelle que soit la chose)
- Suivi du subjonctif.

### Quoique — un mot (= bien que)
- **Quoique** fatigué, il continua.
- Peut être remplacé par "bien que".

## "Quelque...que" / élisions courantes

| Forme | Contexte | Exemple |
|-------|----------|---------|
| quelles que soient | + verbe être, sujet f. pluriel | quelles que soient les arrhes |
| quelque...qu' | + adjectif + que | quelqu'exiguës qu'elles soient |
| quoi qu'il en soit | locution figée | quoi qu'il en soit |
| quelqu'un | pronom | quelqu'un est venu |

## Élisions obligatoires en français
- **de + voyelle** → d' : d'accord, d'abord
- **que + voyelle** → qu' : qu'il, qu'elle, qu'on
- **se + voyelle** → s' : s'écria, s'accrut
- **le/la + voyelle** → l' : l'église, l'imbécillité
- **quelque + voyelle** → quelqu' : quelqu'exiguës (**souvent oublié**)
- **ne + voyelle** → n' : n'est, n'avait
- **me + voyelle** → m' : m'a dit
- **te + voyelle** → t' : t'a répondu

## Locutions conjonctives fréquentes à ne pas confondre

| Locution | Orthographe | Exemple |
|----------|-------------|---------|
| c'est-à-dire | tirets obligatoires | c'est-à-dire que |
| vis-à-vis | tirets obligatoires | vis-à-vis de |
| peut-être | tiret obligatoire | peut-être viendra-t-il |
| par-dessus | tiret | par-dessus tout |
| au-delà | tiret | au-delà de |
| néanmoins | un seul mot | néanmoins |
| cependant | un seul mot | cependant |
