# Règles sur les accents en français

## Accent circonflexe (^)
- **â** : châssis, pâte, mâchoire, infâme, bâtir, bâti, grâce, âge
- **ê** : guêpier, fête, bête, forêt, têtu, même, fenêtre, arrêt
- **î** : dîner, île, maître, paraître, naître, gîte, abîme
- **ô** : ôter, hôpital, hôtel, côte, rôle, diplôme, tôt, côté
- **û** : sûr, mûr, flûte, croûte, voûte, brûler

## Tréma (¨)
- **ï** : naïf, naïveté, héroïque, ambiguïté, exiguïté, contiguïté
- **ë** : Noël, Israël, canoë
- Attention : **ambiguïté** (pas "ambiguité"), **exiguïté**, **contiguïté** — le tréma est obligatoire sur le ï.

## Accent grave (`)
- **à** : préposition (à Paris), distingue de "a" (verbe avoir)
- **è** : mère, père, frère, bélître, guère, procès
- **ù** : uniquement dans "où" (adverbe de lieu)

## Accent aigu (´)
- **é** : porté, été, médecin, bélître, ambiguïté, événement, excédent
- Attention : **événement** (é-vè-ne-ment — accent grave sur le 2e e)

## Invariants d'origine latine et grecque (pluriels et formes)

Ces mots gardent leur forme latine ou grecque et sont souvent **invariables** ou suivent des pluriels irréguliers :

### Mots en -um → -a (pluriel latin)
| Singulier | Pluriel latin (médical/scientifique) | Pluriel francisé |
|-----------|--------------------------------------|-----------------|
| maximum | maxima | maximums |
| minimum | minima | minimums |
| aquarium | aquaria | aquariums |
| forum | fora | forums |
| datum | data | — |
| medium | media | médiums |
| consortium | consortia | consortiums |

### Mots en -us → invariables ou -i (pluriel latin)
| Singulier | Pluriel latin | Note |
|-----------|--------------|------|
| processus | processus | **invariable** |
| campus | campus | **invariable** |
| hiatus | hiatus | **invariable** |
| virus | virus | **invariable** |
| radius | radii / radius | en médecine : radius invariable |
| corpus | corpus | **invariable** |
| syllabus | syllabus | **invariable** |
| stimulus | stimuli | pluriel latin courant |
| nucleus | nuclei | pluriel latin courant |
| calculus | calculi | en médecine (lithiases) |

### Mots en -a → -ae (pluriel latin, usage médical)
| Singulier | Pluriel latin | Note |
|-----------|--------------|------|
| vertebra | vertebrae | anatomie |
| larva | larvae | biologie |
| alga | algae | biologie |
| antenna | antennae | zoologie |
| lacuna | lacunae | médecine |
| fistula | fistulae | médecine |

### Mots en -ex/-ix → -ices (pluriel latin)
| Singulier | Pluriel latin | Note |
|-----------|--------------|------|
| index | indices / indexes | les deux sont acceptés |
| vertex | vertices | anatomie |
| cortex | cortices / cortex | médecine |

### Mots en -on → -a (grec)
| Singulier | Pluriel grec | Note |
|-----------|-------------|------|
| criterion | criteria | philosophie/sciences |

| ganglion | ganglions / ganglia | médecine |

### Termes radiologiques invariables fréquents
- **foramen / foramina** : orifice osseux (ex. : foramen intervertébral → foramina intervertébraux)
- **ostium / ostia** : orifice anatomique
- **septum / septa** : cloison anatomique
- **fundus / fundi** : fond d'organe
- **sinus** : **invariable** (le sinus, les sinus)
- **hiatus** : **invariable**
- **plexus** : **invariable**
- **meatus** : **invariable**
- **ramus / rami** : branche (neurologie, radiologie)
- **sulcus / sulci** : sillon (neurologie)
- **gyrus / gyri** : circonvolution (neurologie)
